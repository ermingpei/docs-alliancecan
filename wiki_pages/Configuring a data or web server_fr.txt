''Page enfant de [[Cloud/fr|Service infonuagique de Calcul Canada]]''

Les instances (machines virtuelles ou VMs) utilisées pour fournir des données génériques ou du contenu Web sont configurées selon les gabarits de type persistant (voir [[Virtual machine flavors/fr|Gabarits d'instances]]) et sont [[Working_with_volumes/fr#Démarrer_depuis_un_volume|démarrées depuis un volume]].

*Pour créer un serveur de données partagées, consultez [[FTP server in the Cloud/fr|Serveurs FTP dans notre environnement infonuagique]] où il est aussi question de HTTP et SFTP.
*Pour créer un serveur web, consultez [[Creating a Webserver on the Cloud/fr|Création d'un serveur web dans notre environnement infonuagique]].
